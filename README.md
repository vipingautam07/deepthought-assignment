# Event & Nudge API (Node.js + MongoDB)

This project is a backend API for managing **Events** and **Nudges** using **Node.js, Express, and native MongoDB driver**.  
It is schema-less, flexible, and designed to support future changes without database migrations.

---

## Tech Stack

- Node.js
- Express.js
- MongoDB (native `mongodb` driver)
- Multer (for image uploads)
